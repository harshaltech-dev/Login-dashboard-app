# Login Dashboard App

React Login Dashboard App with authentication flow, protected routes, React Router navigation, and Local Storage-based user authentication.

## Technologies Used

- React.js
- JavaScript (ES6+)
- React Router
- HTML5
- CSS3
- Local Storage

## Features

- User Login Authentication
- Protected Routes
- Dashboard Navigation
- useState and useEffect Hooks
- useNavigate and useLocation
- Responsive User Interface
